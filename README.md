ShipIt
=============

## nodejs server for ShipIt
* demo here http://www.youtube.com/watch?v=99eVWo1UUSQ

## How to run it
* make sure you have nodejs
* clone the repo
* run `$ npm install`
* run `$ node app.js`
* go to the client repo and follow the instructions [there](https://github.com/retakeio/partgr-client)
